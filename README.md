# PrograII

Alumnas: Escribano, Constanza Agustina; Serra, Sofia; Viand, Sofia Pilar; Zaragoza, Marina Luz.

Actividad 1: Diseño de un sistema de venta de tickets usando una interfaz de cola (FIFO) con operaciones básicas: inicializar, acolar, desacolar, primero y verificar si está vacía.

Actividad 2: Análisis de 3 estrategias de implementación de pila con arreglos, concluyendo que la estrategia 1 (variable externa) es la más eficiente y clara.
